 Carpintería y Arte

Este proyecto busca implementar una aplicación web que apoye el trabajo de carpintería y arte, integrando herramientas digitales para la organización, diseño y colaboración entre estudiantes.


- Aprender y aplicar Git y GitHub en un entorno colaborativo.
- Desarrollar una aplicación web que facilite la gestión de proyectos de carpintería y arte.
- Fomentar el trabajo en equipo y la integración de conocimientos técnicos y creativos.
- Documentar el proceso de desarrollo de manera clara y organizada.

 Público objetivo

 Funcionalidades esperadas
- Registro de usuarios y roles (estudiante/docente).
- Subida de imágenes de proyectos.
- Panel de tareas por equipo.
- Historial de cambios y avances.
- Visualización de ramas y contribuciones en GitHub.
