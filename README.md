# Sistema Web de Gestión para Carpintería & Arte

## Descripción del Proyecto
Este proyecto consiste en el desarrollo de un sistema web para la empresa Carpintería & Arte, dedicada al diseño y fabricación de muebles en madera. 

El sistema tiene como objetivo mejorar la gestión de la información de la empresa, permitiendo administrar de manera organizada los inventarios, pedidos, ventas y cotizaciones.

Actualmente muchos de estos procesos se realizan de forma manual, lo que puede generar errores o pérdida de información. Por esta razón, se propone el desarrollo de una plataforma web que centralice todos estos procesos y facilite el acceso a la información.

## Objetivo General
Desarrollar un sistema web que permita gestionar de manera eficiente la información relacionada con inventarios, pedidos, ventas y cotizaciones dentro de la empresa Carpintería & Arte.

## Funcionalidades del Sistema
- Gestión de inventario de materiales.
- Registro de pedidos de clientes.
- Creación de cotizaciones.
- Control de ventas.
- Consulta de información en tiempo real.

## Tecnologías Utilizadas
- HTML
- CSS
- JavaScript
- Base de datos MySQL
- Git y GitHub para el control de versiones
